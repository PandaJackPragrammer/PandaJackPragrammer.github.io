国学：小六壬

Posted on 2023-09-14，0 min read

各位小伙伴们好呀！这是 PandaJack 博客第一期 AI 周报，从这周开始每周五为各位盘点 AI 圈的重点新闻。先来康康这周有啥好玩的 👇

🔥 Hermes Agent 生态全面开花

这周对 Hermes Agent 用户来说简直是过年！

Hermes Web UI 正式发布（⭐5.7k）

项目地址：https://github.com/EKKOLearnAI/hermes-web-ui

Hermes Agent 终于有了官方 Web 界面！由 EKKOLearnAI 开发的 hermes-web-ui 提供了包括多平台聊天、会话管理、定时任务和用量统计在内的完整功能面板。再也不用全靠终端命令行操作了，界面设计简约清晰，对新手友好度直接拉满。

这个项目自发布以来迅速获得了超过 5700 颗星，增长速度非常夸张，说明社区对 Hermes Agent 图形化管理的需求确实很高。

Hermes Agent 橙皮书中文教程（⭐3.8k）

项目地址：https://github.com/alchaincyf/hermes-agent-orange-book

与此同时，也出现了首个完整的中文 Hermes Agent 深度教程——《Hermes Agent 从入门到精通·橙皮书系列》。这份指南涵盖了从环境搭建、配置管理、工具开发到多 Agent 协作的方方面面，对于中文用户来说是一份非常宝贵的资源。

作者表示后续还会持续更新，覆盖更多高级用法和实战案例。

🌐 AI Agent 新框架和工具

Obscura：AI Agent 专用无头浏览器（⭐13.5k）

项目地址：https://github.com/h4ckf0r0day/obscura

这绝对是本周最重磅的工具发布！Obscura 是一个专门为 AI Agent 和网页抓取设计的无头浏览器，在短短一个多月内就暴涨到 1.3 万星。

与传统的无头浏览器（如 Puppeteer、Playwright）不同，Obscura 从底层开始就针对 AI Agent 的使用场景进行了优化：

- 专为 Agent 设计的 API：可以用自然语言描述操作目标，而不是写繁琐的 CSS 选择器

- 智能等待机制：自动处理动态加载内容，不再需要手动 sleep

- 内存高效：相比 Chromium 系浏览器大幅降低内存占用

对于想做网页自动化 Agent 的朋友来说，这个值得重点关注。

CubeSandbox：腾讯云开源的 Agent 沙箱（⭐5.8k）

项目地址：https://github.com/TencentCloud/CubeSandbox

腾讯云开源的 CubeSandbox 是一个专为 AI Agent 设计的即时、安全、轻量级沙箱环境。亮点在于：

- 秒级启动：不需要等待虚拟机启动

- 完全隔离：每个 Agent 实例运行在独立的沙箱中

- 资源可控：精确限制 CPU、内存和网络访问

这对于需要让 Agent 执行代码或访问外部服务的场景非常实用。

DeepSeek-Reasonix：终端里的 DeepSeek 编码 Agent（⭐5.2k）

项目地址：https://github.com/esengine/DeepSeek-Reasonix

DeepSeek 生态的新成员！DeepSeek-Reasonix 是一个基于 DeepSeek 模型的终端 AI 编码助手，专为前缀缓存（prefix-cache）稳定性做了深度优化，可以长期运行在后台。它支持：

- 代码生成和重构

- 终端命令智能建议

- 持续运行的背景 Agent 模式

有意思的是它的设计哲学——让它一直跑着，就像你的 AI 同事一直在线一样。

📧 Cloudflare Agentic Inbox（⭐3.1k）

项目地址：https://github.com/cloudflare/agentic-inbox

Cloudflare 最近开源了一个很有意思的项目：Agentic Inbox——一个完全运行在 Cloudflare Workers 上的自托管 AI 邮件客户端。它可以：

- 自动分类和回复邮件

- 通过 AI Agent 处理邮件工作流

- 支持自定义规则和过滤

想到之前在博客分享过的各种自部署方案，这个可以无缝接入现有工作流。

💻 本地 AI 部署新动态

Ollama v0.30.0 发布

Ollama 本周推送了 v0.30.0 的候选版本，带来了多项改进：

- 更好的多模态模型支持

- 性能优化和内存管理改进

- 新的模型导入工具

对于在本地部署大模型的朋友来说，Ollama 始终是最便捷的选择之一。值得注意的是，随着 DeepSeek、Qwen 等国产模型的 GGUF 版本越来越成熟，本地部署的门槛正在持续降低。

TencentDB-Agent-Memory（⭐3.8k）

项目地址：https://github.com/Tencent/TencentDB-Agent-Memory

腾讯开源的本地 Agent 记忆系统，为 AI Agent 提供完全本地的长期记忆能力。这意味着 Agent 可以在不上传任何数据到云端的情况下，记住对话历史、用户偏好和任务状态，对于隐私敏感场景非常有价值。

🎯 本周小结

这周的热点非常集中：Agent 基础设施正在快速成熟。从专用浏览器（Obscura）到沙箱环境（CubeSandbox），从 Web 管理界面（Hermes Web UI）到记忆系统（TencentDB-Agent-Memory），AI Agent 所需的各种组件都在快速完善。

另一个趋势是 本地化——越来越多的工具和方案强调本地运行和数据隐私，这对于我们这些喜欢自己折腾的玩家来说是个好消息。
