CapsWriter-Offline：按住说话就打字，完全离线的语音输入神器

Posted on 2026-06-29，5 min read

说到语音输入，大家可能第一反应是手机上的输入法语音键、或者各种云端转写服务。但如果你是个重度文字工作者，用键盘打字速度跟不上脑子的时候，你就会发现——市面上的语音输入方案，要么延迟高、要么必须联网、要么没法自定义。

今天要聊的这个项目，解决了以上所有痛点。

CapsWriter-Offline 是什么

项目地址：https://github.com/HaujetZhao/CapsWriter-Offline

一个完全离线的 PC 端语音输入工具。5800+ Stars，MIT 开源协议。

它的操作逻辑极其简洁：

> 按住 CapsLock 键说话，松开就上屏。就这么简单。

不需要点按钮，不需要切窗口，不需要等网络。按→说→松→字出来，一气呵成。

为什么它值得关注

完全离线，隐私拉满

所有语音识别在本地完成，不需要任何网络连接。识别引擎有四种可选：

| 引擎 | 准确率 | 延迟(20s音频) | 显存加速 |

|------|:-----:|:------------:|:-------:|

| Paraformer | ★★★☆☆ | CPU 0.6s | ❌ |

| SenseVoice-Small | ★★★☆☆ | CPU 0.6s / GPU 0.15s | ✅ |

| Fun-ASR-Nano | ★★★★☆ | CPU 2.0s / GPU 0.5s | ✅ |

| Qwen3-ASR-1.7B | ★★★★★ | CPU 4.0s / GPU 1.0s | ✅ |

最轻量的 Paraformer 和 SenseVoice-Small 在 CPU 上只需要 0.6 秒就能转完 20 秒音频，几乎是实时的。如果你有 N 卡，开启 GPU 预加速后延迟可以进一步压到 0.1 秒级别。

这意味着什么？说的话还没说完，字已经出来了。

无感输入体验

CapsWriter 的设计哲学是"如臂使指"——让你感觉不到它存在。

- 默认模式：按住 CapsLock 说话，松手自动上屏，自动去掉末尾逗句号

- 对讲机模式：按住就录，松手就上屏

- 单机录音模式：点一下开始录，再点一下结束上屏

配合托盘菜单，可以快速开关、切换模式、添加热词，全程不需要离开当前窗口。

高度自定义

这是 CapsWriter 和普通语音输入最大的区别——它的热词系统非常强大：

- hot.txt：写下偏僻词（人名、专业术语、英文缩写），通过音素模糊匹配自动替换

- hot-rule.txt：用正则表达式或简单等号规则做精准替换

- 自定义短语：说"我的地址" → 自动展开为完整地址

对于程序员、医生、律师、翻译这些有大量专业术语的人群来说，这个功能直接把语音输入的可用性拉高了一个档次。

LLM 角色润色

v2.6 版本开始，内置了 LLM 润色功能。

你可以在识别结果开头用特定词触发角色处理，比如说"润色：今天天气真不错"——结果会先经过 LLM 润色再上屏。目前支持 DeepSeek API，配置好密钥就能用。

虽然需要网络调用 LLM（这不是语音识别的一部分，是额外功能），但你可以选择不用，语音识别本身全程离线。

文件转录：丢进去就出字幕

除了实时语音输入，CapsWriter 还支持文件转录。把音频或视频文件拖到客户端窗口上，自动输出：

- .srt 字幕文件

- .txt 文本文件

- .json 时间戳文件

做播客笔记、会议录音转文字、视频字幕制作，一条龙搞定。

一些我实际用过后的感受

说说实际体验。这个项目最打动我的是它的设计取舍：

1. 离线优先 — 不给自己留联网的后路，把本地模型优化到能用的水平。这是一种信念感：离线不是妥协，而是设计的出发点。

2. 按→说→松 — 这个交互太直觉了。不需要学习，不需要记忆，甚至不需要看。肌肉记忆一旦形成，打字效率翻倍。

3. 克制 — 它不做图形界面、不做 Mac/Linux 支持（作者明确说只专注于 Windows），反而让它更好用了。功能少而精，一个 U 盘就能带走，插哪用哪。

当然，也有遗憾：

- 仅 Windows：Mac 和 Linux 用户无缘了（作者推荐了 LazyTyper 和闪电说作为替代）

- 需要一点动手能力：下载模型、配置热词、调整参数，不是开箱即用那种直接双击就行的应用

- Qwen3-ASR 模型较大：最高的准确率需要 1.7B 模型，4G 显存起步

适合谁用

- 作家 / 编辑 / 自媒体 — 码字量大，语音输入能解放双手

- 程序员 — 写文档、写周报、记笔记，CapsLock 键本来就是日常按的

- 会议密集人士 — 快速记录想法，事后整理

- 隐私敏感用户 — 所有数据不出本机

- 手部不适的用户 — 减少键盘依赖

总结

CapsWriter-Offline 在 5.8k Stars 的体量下，质量出乎意料地高。它不追求"智能"、"AI 赋能"这些花里胡哨的概念，而是死磕一个点——让你说话代替打字的体验无限接近零摩擦。

如果你在 Windows 上工作、打字量大、又在意隐私，这可能是目前最优的离线语音输入方案，没有之一。
